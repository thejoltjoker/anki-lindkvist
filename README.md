# Anki Lindkvist



Beautiful Anki card templates with example Italian vocabulary decks. This repository contains two distinct card templates for different learning styles.

## 📋 Templates

### [Lindkvist Template](lindkvist/README.md)

Traditional cloze deletion cards with clean, modern styling for focused vocabulary learning.

### [Brolingo Template](brolingo/README.md)

Interactive multiple-choice cards with mascot character and gamified learning experience.

## 🗂️ Project Structure

```
lindkvist/              # Traditional cloze deletion template
├── card/               # Template files
└── example/            # Example data and prompts

brolingo/               # Interactive template
├── card/               # Template files with JavaScript
└── example/            # Example data and prompts
```

## 🚀 Quick Start

1. **Choose your template:**

   - **Lindkvist**: For traditional cloze deletion learning
   - **Brolingo**: For interactive, gamified learning

2. **Follow the template-specific README:**

   - See `lindkvist/README.md` for Lindkvist setup
   - See `brolingo/README.md` for Brolingo setup

3. **Try the examples:**
   - Both templates include example Italian vocabulary decks
   - Use the provided AI prompts to generate your own content
